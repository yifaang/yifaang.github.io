论如何使用Git进行代码托管

安装Git

在Linux下自然是不用多说 apt install git 即可，作为Windows用户的话，自行寻找安装方法！

Git配置

对需要进行上传的文件先进行clone

然后进行配置私钥，Git在某年的情人节中早就禁止使用账号密码登录了，所以只能使用公私钥进行登录了

生成私钥[Setting]->[Developer Setting]->[Personal access tokens]->[Generate new token]

生成后的token只会出现一次，请妥善保存

然后对Git进行配置,

简单的一比就离谱！
